# AWS Cognito Login System using EC2, Apache, PHP & HTTPS

## Project Overview

This project demonstrates a secure authentication system built using Amazon Cognito Hosted UI and deployed on an AWS EC2 instance running a LAMP stack.

The application allows users to authenticate using Amazon Cognito, securely exchanges OAuth2 authorization codes for tokens, creates user sessions in PHP, and displays a protected dashboard.
